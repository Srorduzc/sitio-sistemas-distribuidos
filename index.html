¿Qué son los Sistemas Distribuidos?

Un sistema distribuido es un conjunto de computadoras independientes que trabajan juntas y se presentan ante el usuario como un único sistema coherente. Estas máquinas se comunican a través de una red y coordinan sus acciones para compartir recursos, procesar información y ofrecer servicios de manera eficiente.

Entre sus principales características se encuentran la escalabilidad (capacidad de crecer), la tolerancia a fallos (seguir funcionando aunque fallen algunos componentes) y la transparencia (el usuario no percibe la complejidad interna del sistema).

Ejemplos de sistemas distribuidos incluyen plataformas en la nube como AWS, Azure o Google Cloud, redes sociales, servicios de streaming y sistemas de bases de datos distribuidas.
