# sitio-sistemas-distribuidos
Sitio web sobre Sistemas Distribuidos - Actividad práctica


### 1. ¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?

Permite acceder al proyecto desde cualquier lugar sin instalar nada, facilita el trabajo colaborativo, guarda versiones automáticamente y evita perder información.



### 2. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?

Dificultades para editar archivos (errores al guardar), menos control sobre el entorno, posibles problemas de conexión y limitaciones del editor web.



### 3. ¿Cómo podrías mejorar este sitio web en el futuro?

Agregar interactividad con JavaScript, mejorar el diseño visual, hacerlo más responsive, incluir backend para procesar el formulario y desplegarlo en un servicio en la nube.



### 4. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?

Que permiten trabajar desde distintos lugares en un mismo proyecto, manejar versiones de forma distribuida y entender cómo múltiples sistemas se coordinan (como GitHub en la nube).
